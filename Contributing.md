# Contributing to Bubble Tea (Python)

Thank you for your interest in contributing! This is a Python port of the 
[Go Bubble Tea library](https://github.com/charmbracelet/bubbletea).

## Getting Started

1. Fork and clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # or `venv\Scripts\activate` on Windows
   ```
3. Install development dependencies:
   ```bash
   pip install -e ".[dev]"
   ```

## Development

### Running Tests

```bash
pytest
```
